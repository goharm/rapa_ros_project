개발은 각 단위별 모듈로 개발한다.

+ 개발 예정 모듈은 다음과 같다.
    + 구동부
        - 선회 -> turning.py
        - 전진 -> comming.py
        - 도리도리 -> nonono.py
        - 지그재그 후진 -> zigzag_back_off.py
    + 딥러닝
        - "이리와" -> gamestart.py
        - "안놀아" -> gameover.py
        - 안면인식 -> whois.py
        - 가위바위보 게임 -> game.py
    + 머신러닝(cv)
        - 동제권자와 로봇간의 거리 -> space_check.py

각 모듈별 개발 내용은
메인 프로그램에서 호출하여 활용한다.
